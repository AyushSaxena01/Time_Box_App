# Time Box App

A simple, focused web application for dividing your available time into focused work segments and tracking progress through each segment.

## Overview

Refer to the detailed application documentation here:
➡️ **[time-box-app/README.md](time-box-app/README.md)**

This project demonstrates a **spec-driven development approach**, where specifications are written before implementation. This ensures that requirements, architectural decisions, and validation criteria are clearly defined upfront.

The project was developed using **VS Code** with **GitHub Copilot** to assist in implementation based on the defined specifications.

---

## OpenSpec Framework

This project follows the **OpenSpec** framework, a spec-driven development approach that provides:

* **Structured change management** – Changes are proposed, designed, specified, and then implemented
* **Artifact-driven workflow** – Each change generates multiple artifacts (proposal, design, specifications, tasks)
* **Clear separation of concerns** – Specifications drive implementation, and implementation validates the specifications
* **Hierarchical specifications** – Individual capability specifications support higher-level change specifications
